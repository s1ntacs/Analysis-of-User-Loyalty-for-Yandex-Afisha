# Analysis-of-User-Loyalty-for-Yandex-Afisha
Работа над итоговом проектом спринта яндекс практикума 

#Исследовательский анализ данных платформы бронирования билетов с целью выявить факторы, влияющие на повторные покупки.  
На основе данных построены профили пользователей, проанализированы:  
- поведение по типу мероприятия, устройству, региону и оператору,  
- влияние выручки, количества билетов и частоты заказов на удержание,  
- временные паттерны активности.  

Выявлены ключевые сегменты лояльных клиентов и даны рекомендации по повышению удержания.  

Проект выполнен на Python (`pandas`, `seaborn`, `phik`) с подключением к PostgreSQL.
